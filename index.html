What my project?

Project
Follow-me-carrier

The hardware platform design utilizes the ARDUINO UNO R3 module as the primary control module, complemented by the motor driver module (L298N) and the Bluetooth module (HC_05) as essential components. For software design and development, Arduino programming is utilized to establish connections and control between these modules. Additionally, it enables the reception of instructions from an Android application. To facilitate the auto-following mode, the turning of the carrier is accomplished using the gyroscope sensor present in the mobile phone. Furthermore, the distance between the carrier and the mobile device is achieved through the utilization of the Bluetooth GATT API.

Keywords: Arduino programming, Android programming, Bluetooth signal strength analysis, Auto-following mode

Android APP development

To test the functionality of the sensors in a mobile phone, namely the accelerometer, gyroscope, and proximity sensor, the app utilizes radio buttons to display each function individually. Additionally, the project utilizes the Bluetooth connection APIs provided by the Android platform to establish a connection between two phones, enabling the transmission of data between them.

Keywords: Java, Sensors controlling, Bluetooth connection service

Caligraphy Generator

The project offers a graphical user interface (GUI) that allows users to generate icons featuring Chinese calligraphy. The method employed in this project involves performing web crawling to obtain Chinese calligraphy figures. Additionally, the GUI interface provides customization options, allowing users to personalize their figures.

Keywords: Python, GUI interface, Web crawler
